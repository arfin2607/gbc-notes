# COMP1238 - Data Management

Course notes for Data Management course at George Brown College.

## Week 1 - History of Computing
- Early computers used vacuum tubes and took up entire rooms
- The first programmer was Ada Lovelace who worked on Charles Babbage's Analytical Engine
- ENIAC was one of the first general-purpose electronic computers

## Week 2 - Number Systems
- Binary system uses only 0 and 1 to represent all data
- Hexadecimal (base-16) is often used as a shorthand for binary
- Converting between binary, decimal, and hexadecimal is essential for understanding how computers store data

## Week 3 - Data Storage
- Bits are the smallest unit of data (0 or 1)
- Bytes are 8 bits and can represent 256 different values
- ASCII uses 7 or 8 bits to represent text characters
- Unicode allows for representation of characters from all writing systems

## Week 4 - Data Organization
- Files are organized in hierarchical directory structures
- File extensions indicate the type of data in a file
- Metadata is data about data (creation date, file size, permissions)
- Different file systems (FAT32, NTFS, ext4) organize data differently on storage devices.

## Week 5 - Database Concepts
- Databases store organized collections of data
- Tables contain rows (records) and columns (fields)
- Primary keys uniquely identify each record in a table
- Foreign keys create relationships between tables
